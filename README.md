# 数据安全检查工具 Data Safety Scanner 🔍

## 简介

扫描本地文件中的个人敏感信息，用于数据安全检查。完全离线运行，无需网络。

## 功能

**扫描识别：**
| 敏感信息类型 | 风险等级 | 示例 |
|---|---|---|
| 身份证号 | 🔴 高 | 110101199001011234 |
| 手机号 | 🔴 高 | 13800138000 |
| 银行卡号 | 🔴 高 | 622202xxxxxxxxxxxx |
| 电子邮箱 | 🟡 中 | name@company.com |
| 固定电话 | 🟡 中 | 010-12345678 |
| IP地址 | 🟢 低 | 192.168.1.1 |
| API密钥/Token | 🔴 高 | api_key=sk-xxx... |
| 护照号 | 🔴 高 | E12345678 |
| 地址关键词 | 🟡 中 | 含省/市/区等 |

**操作处理：**
- ✅ 查看匹配上下文（前后20个字符）
- 📂 打开文件所在位置
- ✂️ 脱敏处理（将敏感信息替换为 `***`，自动备份原文件为 `.bak`）
- 🗑️ 删除文件（移入废纸篓）
- 📄 导出扫描报告（JSON / 文本格式）

## 使用方法

### 方式一：双击启动（推荐）
1. 进入 `数据安全检查工具` 文件夹
2. 双击 `启动.command`
3. 点击「添加目录」选择要扫描的文件夹
4. 选择需要检测的敏感信息类型
5. 点击「开始扫描」
6. 扫描完成后，右键点击结果条目进行操作

### 方式二：命令行启动
```bash
cd /Users/feng/Desktop/Hermes文件/数据安全检查工具
python3 DataSafetyScanner.py
```

## 注意事项

1. **扫描范围：** 支持 .txt/.md/.csv/.log/.json/.xml/.yaml/.py/.html/.js/.env 等文本格式；不支持图片、PDF扫描件、加密文件
2. **大文件跳过：** 默认跳过 >10MB 的文件，避免卡顿
3. **误报提示：** 正则匹配可能存在误报，建议逐条核查后再删除
4. **删除操作：** 删除的文件移入macOS废纸篓，可恢复
5. **脱敏操作：** 脱敏前会自动备份原文件为 `.bak`

## 运行环境
- macOS / Linux / Windows（需Python 3.6+）
- Python标准库（Tkinter），无需额外安装依赖
